# プライバシーポリシー

最終更新日: 2026年1月23日

KEY Planning co.,ltd.（以下「当社」）は、iOS アプリ「Cursory」（以下「本アプリ」）におけるユーザー情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

## 1. 収集する情報

本アプリは、サービスの提供および機能向上のため、以下の情報を収集・送信します。

### ユーザーコンテンツ
- **テキスト情報**: ユーザーが入力したエージェントへの指示（プロンプト）、会話内容
- **画像情報**: ユーザーが任意で添付した画像（スクリーンショット等）

### アカウント・認証情報
- **Cursor API キー**: エージェントを起動・管理するため
- **GitHub アカウント情報**（任意）: リポジトリの参照や PR へのアクセスのため

これらの情報は、本アプリの機能（Cursor Cloud Agents API および GitHub API へのリクエスト）を提供するためにのみ使用され、**ユーザーに関連付けられたデータ**として取り扱われます。

## 2. 利用目的

収集した情報は、以下の目的でのみ利用します。

- **アプリの主要機能の提供**: エージェントの操作、チャット、画像解析、ソースコード管理
- **認証**: 外部サービス（Cursor, GitHub）への接続

本アプリは、**トラッキング（追跡）**や**サードパーティ広告**のためのデータ収集は行いません。

## 3. 情報の保存と保護

- **API キー / トークン**: iOS のセキュアな保存領域（Keychain）に暗号化して保存されます。
- **通信の暗号化**: すべての通信は HTTPS（TLS）を用いて暗号化されます。

## 4. 第三者への提供

本アプリは、機能提供のために以下の外部サービスへAPIリクエストを行います。これ以外の第三者へ個人情報を販売・共有することはありません。

- **Cursor (Anysphere)**: エージェント機能の実行
- **GitHub**: リポジトリ情報の取得、PR作成など
