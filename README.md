# Cursory

**スマホで Cursor Cloud Agents を操作**

Cursory は、Cursor の Cloud Agents API を活用した iOS ネイティブアプリです。スマートフォンからリポジトリ上で動作する AI エージェントを起動・管理し、開発フローを加速させます。

</a>

## 主な機能

- 🤖 **エージェント管理**: エージェントの起動、停止、一覧確認
- 💬 **会話とフォローアップ**: 進捗ログの確認と、追加指示（フォローアップ）の送信
- 📸 **画像添付**: スクリーンショットを添付してUI修正を指示
- 🔗 **GitHub 連携**: リポジトリの選択や作成された PR へのアクセス
